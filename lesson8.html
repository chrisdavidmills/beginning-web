Floats and CSS layouts

As you'll see when you look at different websites, most of them don't have a single column of text going from top to bottom. Most of them have multiple columns of text sitting alongside one another. As examples, let's look at The Mozilla Developer Network, The BBC home page, and as a simpler example, my Fansite template. So how do we do this?

Introducing floats

There is a property available in CSS called float, which can take a value of left or right. When you set this on an element, it causes that element to float downwards around the left or right edge of the element below it in the source order. Originally this was designed to allow us to have images floating inside nearby blocks of text, in the same way that you might have in a newspaper or magazine article. A simple CSS layout

Even though floats were only really intended for simple floated images in the first place, you can apply a float to anything you want. Web developers very quickly realised that they could start using floats to create multiple column layouts, which was pretty cool.

Let's look at a typical multiple column layout example, to get us familiar with the concept.

  1. Start a new Thimble page.
  2. In this page's body, insert the following HTML:
    3. <div class="header"><h1>This is my page heading</h1></div>
<div class="main-column"><p>This is my main column.</p></div>
<div class="side-column"><p>This is my sidebar.</p></div>
<div class="footer"><p>My page Â© 2014.</p></div>
  3. We've now got four container elements with a bit of content inside each one. Let's highlight the area they take up and make them a bit taller, so we can see what is going on more easily. Create a style element inside the document head, as before.
  4. Add the following CSS into the style element:
    5. .header, .footer {
  background-color: green;
  height: 100px;
}

.main-column, .side-column {
  height: 200px;
}

.main-column {
  background-color: blue;
}

.side-column {
  background-color: yellow;
}
  5. Now let's go about creating a two column layout. We need to alter the width of the main column and side column so they will fit alongside one another. Then we need to make them both float so they sit alongside one another rather than being on top of one another. Update the bottom two rules so they look like this:
    6. .main-column {
  background-color: blue;
  width: 58%;
  float: left;
}

.side-column {
  background-color: yellow;
  width: 40%;
  float: right;
}

Clearing floats to solve problems

So now we have a two column layout with a gutter in between, but all is not well. The green colour from the footer has been slurped up into the gutter, and the footer content is left looking a shabby colourless mess down below. This is because when you tell an element to float, it will float down the page forever, well, until the bottom of the element. Everything than can, will float around it, including our footer. They way to deal with this is to use clear a property which when set on an element basically says "stop floating here."

clear takes one of three values: left, right or both. Which one you use depends on whether you want to stop left floats happening, right floats, or left and right at the same time. I usually find that both is the best to use, to be an the safe side. To fix the problem in this case, add the following rule to the bottom of your CSS:

.footer {
  clear: both;
}

This should fix your layout, giving you a look like the screenshot below.
