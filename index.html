It's my web!

A ten week course to teach the basics of web design

Chris Mills, Mozilla

Introduction

The Web is a good place to begin learning about how technology works, and fundamental Information and Communication Technology (ICT) concepts such as programming. It is everywhere and familiar, easy to learn compared to other programming platforms, and there is a lot of help available due to the Web's largely free and open nature. This course will aim to teach children some background information on how the web works and why the web is so interesting, the basic fundamentals of HTML (the language used to structure data on the Web) and CSS basics (the language used to style and layout the Web), possibly then moving on to some more advanced CSS and JavaScript (the web's main programming/logic language) if there is time.

This course has mainly been written with 10-11 year old children in mind, but anyone who is a complete beginner to web design should be able to get a lot out of it.

Note: if you have any feedback or change requests you want to give about the course, send pull requests/comments to the Introduction to Coding Github repo, or send Chris a message on Twitter.

What do you need?

To take this course, you need:

  • These notes, either in printed or online form. Online is better, as there are a lot of linked resources in here that won't work very well on paper
  • A computer with Internet access
  • At least one decent web browser installed: Any of the following are ok:
    • Firefox
    • Chrome
    • Safari
    • Opera
    • Internet Explorer (10 or above)
  • A Mozilla Webmaker login (you basically just need an e-mail address.) If you can't or don't want to get this, JSBin is a good alternative.

Contents

The ten documents referenced below contain ten lessons, to be taught over ten, one hour-long sessions.

  • Lesson 1: What is the Web, really? Why is it interesting?
  • Lesson 2: Getting started with code and projects
  • Lesson 3: HTML basics
  • Lesson 4: More HTML to get stuck into
  • Lesson 5: CSS basics
  • Lesson 6: Fundamentals of how CSS works
  • Lesson 7: CSS decorations
  • Lesson 8: CSS layout and free time
  • Lesson 9: Positioning and free time
  • Lesson 10: How do you become a web developer?
