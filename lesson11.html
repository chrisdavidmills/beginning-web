In this article we will explore some supplementary techniques that aren't exactly essential for web development, but are very useful for web users in general.

Searching for images on the Web

At the bottom of lesson 1 — the Hacking the school website section — we discussed how to find an image using Google image search. In this section we will expand this topic further.

Copyright

One thing you need to bear in mind on the Web when you start to use other people's images (or other types of files) in your own work is copyright. When people publish work, they publish it under a license that specifies what rights other people have to use it.

If the license is "All rights reserved", it means that legally you can't use their work unless you make a deal with them, which usually involves paying them some money.

However, there are many licensing systems that do permit reuse of work, which some conditions, such as:

  • You need to publish your work under the same license as the work you reused.
  • You can't reuse the work for a commercial project (i.e. one that will be sold.)
  • You need to credit the author of the work you reused.

The conditions differ from license to license. For a good guide to the most common license types, read A Short Guide To Open-Source And Similar Licenses, by Cameron Chapman.

Finding open source licensed images using flickr

Flickr is an image sharing service, so great for finding images. To search for open source licensed images, do this:

  1. Go to Flickr's Advanced search page.
  2. Type what you want to search for in the search box at the top.
  3. Check the Only search within Creative Commons-licensed content and Find content to modify, adapt, or build upon boxes.
  4. Click Search.

Clever tricks with Google search

Google are still best known for being a search engine, but you might be interested to know that there are a number of clever tricks built into the search bar. Let's investigate a few now.

Basics: Complete strings, wildcards, or excluding words/sites.

  • By default, Google will search for the string of text you give it, but not in the exact order, or all together. To make sure it searches for a complete string of text in that exact order, put quotes round it — example: "my dog has no nose".
  • If you want to search for a phrase but are unsure of one or more words, you can include a * character as a wildcard in place of the unknown words, and Google will give you suggestions — example: how to * a phone.
  • If you want to search for something, but exclude a word, you can put a minus before that word (no space). Example: optimus prime -galvatron.
  • You can also do the same things with whole websites — example: game reviews -site:gamespot.com or film news site:bbc.co.uk

Searching inside web addresses or site titles

  • If you want to search for something just inside the web page title, you can put -intitle: before it — example: -intitle: dinosaur (note the space after the :)
  • If you want to search for something just inside the web address (URL), you can use -inurl: instead — example: -inurl: topgear (note the space after the :)

Finding specific filetypes and locations

  • You can search for specific filetypes using filetype: — example XBox One filetype: pdf (note the space after the :).
  • You can also find sites associated with specific locations, using location: — example hotels location: auckland

Number ranges and maths

  • You can search between a range of numbers using two dots between them. examples: wars 1900..1950, camera £200..£400
  • You can also do maths in the Google search bar. Try these:
    • 9+30
    • 20-4
    • 30*30 or 30x30
    • 40/10
    • 4^2

Reverse image search

If you have an image but you don't know what it is and want to find out:

  • Go to Google image search.
  • Click the little camera icon in the search bar.
  • Now you've got two options - you can either paste in the web address of an image, or upload an image you have on your local hard drive. Here are some examples to try it out on if you've not got any of your own: Example 1, Example 2.

Advanced: adding parameters to the search query string

When you do a search in Google, you'll notice a load of stuff put on the end of the search result web address. These are called parameters, and they give Google extra information on how to search for things. You can hack these on the end yourself. Here are a couple of examples.

  • To search images just for faces, put &tbs=itp:face on the end — example Search for Rose in google image search, then put &tbs=itp:face on the end of the web address and press enter.
  • To search for recently updated pages, you can put for example &tbs=qdr:h5 to search for results updated in the last 5 hours, or you can replace the h with m or s for 5 months or seconds. You can also replace the number with anything you like. Example — search for politics site:bbc.co.uk in normal Google, then put &tbs=qdr:h1 on to the end of the web address and press enter..

Google Easter eggs

To show they are not completely without a sense of humour, Google have also included several easter eggs. Try searching for these:

  • do a barrel roll
  • askew
  • google in 1998
  • atari breakout (in google image search)
  • set timer for, followed by an amount of time, e.g. 10 seconds.
  • bacon number followed by a film star's name.
  • answer to life, the universe and everything
  • Are you a pirate? Search with Google Pirate instead.
  • My favourite: search for police telephone box, click on the first search result and select street view.
